Steam

Reads your achievement unlock dates and fills in missing dates on games you already have here: the earliest unlock becomes the start date, the latest becomes the completion date. Nothing already filled in is overwritten, and no new games are added. Your Steam profile and Game details must be set to Public.

Free from steamcommunity.com/dev/apikey. Stored on your server, never shown again.

PlayStation Network

Same idea, using your trophy earned-dates. Heads up: Sony has no public API, so this uses the same internal endpoints the PlayStation site does, authenticated with an NPSSO token — effectively a session credential for your account. It's stored on your server and never sent back to the browser, but it is unofficial and can stop working whenever Sony changes things.

Log in to PlayStation, then open ca.account.sony.com/api/v1/ssocookie and copy the 64-character npsso value. It expires periodically — just paste a fresh one.

Completion times (IGDB)

Fills in the hours-to-beat for games that don't have one yet. Leave this blank and it's simply skipped.

Note: Metacritic can't do this — it publishes scores and reviews, not completion times, and its only official API is paid/enterprise. HowLongToBeat has no public API and blocks automation. IGDB (owned by Twitch/Amazon) is the one source with an official, free time-to-beat endpoint, so it's what this uses.

Create a free app at dev.twitch.tv/console/apps to get both.
